Instalación y Configuración de Proxmox VE 8

Se instaló y configuró el hipervisor Proxmox VE 8 en el servidor físico, asegurando su operatividad completa.

Segmentación de Red y Zonas

Infraestructura segmentada en tres zonas: DMZ, LAN y WAN. Cada zona está aislada y configurada correctamente.

Contenedores LXC en la DMZ

  • Servidor DNS - IP: 10.0.10.10
  • Sitio Git personalizado - IP: 10.0.10.11
  • Servidor Nextcloud - IP: 10.0.10.13

Despliegue de Máquinas Virtuales

Máquinas virtuales implementadas:

  • Gateway (GW) - LAN: 192.168.56.10, DMZ: 10.0.10.1, WAN: asignada
  • Servidor Zentyal (ZAD00) - IP DMZ: 10.0.10.12

Integración de Servicios

Integración entre Zentyal y Nextcloud completada. Se permite acceso con usuarios nombre.apellido y alumno.upds.
